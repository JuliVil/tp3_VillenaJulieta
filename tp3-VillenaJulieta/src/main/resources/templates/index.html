Variables Miembro

En Java es posible definir variables fuera del ámbito de un método. A estas variables se las denomina como variables miembro. La caracteristica principal de este tipo de variables es que son variables globales dentro de una clase. Por lo tanto pueden ser utilizadas por los métodos de esta clase.

Modificadores de Acceso

Los modificadores de acceso determinan desde que clases se puede acceder a un determinado elemento, como una variable miembro o un método.

Existen 4 tipos:

  • Nivel de acceso por defecto
  • Nivel de acceso public
  • Nivel de acceso private
  • Nivel de acceso protected
